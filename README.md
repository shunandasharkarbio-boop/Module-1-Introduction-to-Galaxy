# Module 1: Introduction to Galaxy

A beginner-friendly introduction to the Galaxy platform for bioinformatics data analysis and workflow execution. This project explores the Galaxy interface, basic workflow operations, and simple sequencing data analysis through hands-on practical exercises.

## Technologies

* Galaxy Platform
* Bioinformatics Workflow
* NGS Analysis
* GitHub

## Learning Outcomes

* Galaxy interface navigation
* Workflow-based analysis
* Basic bioinformatics concepts
* Scientific data processing
